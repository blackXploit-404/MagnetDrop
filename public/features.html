What is MagnetDrop?

MagnetDrop is a peer-to-peer (P2P) file-sharing service that allows users to share files instantly using WebTorrent technology. Unlike traditional cloud storage, your files are **never uploaded to a server**—they are streamed **directly between users**.

Frequently Asked Questions

1. Does MagnetDrop use the torrent network?

Yes, MagnetDrop uses WebTorrent, which allows files to be shared using **torrent-like** technology but without public trackers.

2. Can anyone download my file?

No, only people who have the **magnet link** can access the file. The file is not stored on any server, and it disappears once the original uploader closes their browser.

3. How does MagnetDrop transmit files?

Files are **split into small chunks** and transmitted **peer-to-peer (P2P)** using **WebRTC**. This allows real-time streaming without needing a central server.

4. Is there any risk of file leakage?

No, files are **not discoverable** on public torrent networks. However, **anyone with the magnet link can download the file**, so users should share links carefully.

5. What happens if I close my browser?

If you are the only seeder, the file becomes **unavailable**. If others are downloading it, they may continue seeding it after you leave.
